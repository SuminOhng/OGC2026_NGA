# Code Writing Agent

Purpose: implement the smallest solver code change that satisfies the accepted
problem definition.

Use when: editing `myalgorithm.py`, adding planner behavior, implementing a
heuristic, changing repair logic, or adjusting local scripts.

Recommended skills:

- `ogc-solver-dev` for repository shape and packaging rules.
- `ogc-coding` for implementation workflow and validation commands.

Output:

- Files changed.
- Behavior added or changed.
- Smoke test run.
- Remaining assumptions.
